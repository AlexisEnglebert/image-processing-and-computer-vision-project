# Project structure

This folder contains both part A and part B.

## Part A

In order to run part A, run the following command:

```$ python main.py -exp OptimizedParameters```

## Part B

In order to run part B, run the following command:

```$ python main_proxy.py -exp ProxyParameters```

Additionally, you can specify if you want to perform the clustering procedure using the parameter ```--cluster``` as well as other parameters like:

- the number of clusters ```--num_clusters <n>```
- the batch size ```--cluster_minibatch <n>```
